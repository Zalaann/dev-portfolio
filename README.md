## Installation

1. Clone the repository:
```bash
git clone https://github.com/Zalaann/My-Portfolio.git
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

- `/src/app` - Next.js pages and layouts

- `/src/contexts` - React context providers
